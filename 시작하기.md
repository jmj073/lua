## ⚡ 빠른 참조

### 주요 매크로
```c
CREATE_ABCk(op, a, b, c, k)  // 대부분의 명령어
CREATE_ABx(op, a, bx)         // LOADK, CLOSURE
CREATE_sJ(op, a, sj)          // JMP
SETARG_sJ(inst, sj)           // 점프 패치
```

### 토큰 타입
```c
TK_FUNCTION  // fn
TK_LOCAL     // local
TK_IF        // if
TK_ELSE      // else
TK_ELSEIF    // elseif
TK_RETURN    // return
TK_NE        // !=
TK_DOTS      // ...
TK_TBS       // {|
TK_TBE       // |}
```

### VM 명령어 (자주 사용)
```c
OP_MOVE       // R[A] := R[B]
OP_LOADK      // R[A] := K[Bx]
OP_LOADNIL    // R[A] := nil
OP_GETTABUP   // R[A] := UpValue[B][K[C]]
OP_SETTABUP   // UpValue[A][K[B]] := RK(C)
OP_GETTABLE   // R[A] := R[B][RK(C)]
OP_SETTABLE   // R[A][RK(B)] := RK(C)
OP_NEWTABLE   // R[A] := {}
OP_CALL       // R[A](...) with results
OP_RETURN     // return R[A], ... ,R[A+B-2]
OP_JMP        // pc += sJ
OP_TEST       // if (not R[A] == k) then pc++
OP_CLOSURE    // R[A] := closure(Proto[Bx])
OP_VARARGPREP // vararg setup
```

## 🎓 핵심 개념

### AST 구조
```
AstNode (header)
  ├─ kind: AstNodeKind
  └─ line: int

Expression Nodes
  ├─ AstNumeral, AstString (literals)
  ├─ AstVar (변수)
  ├─ AstBinOp, AstUnOp (연산자)
  ├─ AstCall (함수 호출)
  ├─ AstTableCtor (테이블 생성)
  └─ AstFunction (함수 정의)

Statement Nodes
  ├─ AstReturn (반환)
  ├─ AstLocalDecl (지역 변수)
  ├─ AstAssignment (할당)
  ├─ AstIf (조건문)
  └─ AstFunctionStmt (함수 문)
```

### 코드 생성 흐름
```
gen_chunk()
  └─ gen_statlist()
      └─ gen_statement()
          ├─ gen_return()
          ├─ gen_if()
          ├─ gen_local_decl()
          └─ gen_assignment()
              └─ gen_expr()
                  ├─ gen_binop()
                  ├─ gen_unop()
                  ├─ gen_call_expr()
                  ├─ gen_function()
                  └─ gen_table_constructor()
```
